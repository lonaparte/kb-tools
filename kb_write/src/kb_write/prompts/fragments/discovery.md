## Required reading, in this order

1. **`AGENT-WRITE-RULES.md`** — normative rules for all writes. Non-negotiable.
2. **`.agent-prefs/README.md`** — index of the user's stored preferences, then
   every `.md` file under `.agent-prefs/`. Those encode the user's quirks,
   style rules, and research context that apply across sessions.
3. **`README.md`** — directory map and tool overview.
